
# Samachar App
A Simple News App build using Kotlin.in this app there is a HomePage which display the top Headlines from NewsApi.org.You can Search the News and Saved the News Offline to Read.



## Library Included:
- Navigation Component
- Glide
- Room DataBase
- Retrofit
